# Description

This is a covid-19 dataset. There are two datasets.

First dataset, covid_data.csv, contains data like country, age, sex, lives in wuhan, symptoms, chronic diseases, and the outcome (death, recovered, etc) for the people affected.

The second dataset, covid_outcome.csv, contains information on Confirmed cases, deaths and recoveries in various countries.

Analysis strategy¶

For the First dataset, I aim to analyze the types of symptoms that the people infected by Covid-19 suffered. For this, I will have to segregate the text in symptoms into individual symptoms and map it to a new column in the dataset.

We will also look at the average age group for people who survived the COVID-19 infection and those who could not. We will see how many people visited Wuhan (directly or traveled via Wuhan) and were tested positive for Corona Virus. We will look at people battling some of the chronic diseases and the outcome in terms of recovered, death, sever, hospitalized, or discharged from the hospital.

Using the second database, we will look at the countries with the maximum number of cases confirmed. We will also look at top 5 countries with the maximum number of deaths recorded.

To begin any analysis in the first dataset, we need to clean the following columns:

  1. The ID column has empty values. Resetting the column so that later data can be selected using Index. Then, set it as the index column.

  2. Clean Column 'AGE': Replace Dates with Age Irrelevant Entry 'Belgium' in age column Replace null values with 0 Replace entries with range like 30 - 40 with average average of the two numbers

  3. In symptoms columns, for values like (fever 37.1 C), the parenthesis has to removed as they are causing error in further processing.

    Correcting spelling Mistakes.

    And correcting entries to treat body pain and body ache, running nose and runny nose, respiratory complaints, respiratory distress, respiratory problems lungs, respiratory problems, and respiratory stress as respiratory symptoms. And so on.

  4. Clean Column Outcome Replacing: Died and dead with death. Discharge with Discharged. Intensive care and critical condition with sever. Released from quarantine and Stable with recovered. Recovering at home 03.03.2020 with hospitalized. Recovering at home 03.03.2020 with hospitalized.

    For the record with a date_admission_hospital, filling the outcome column with 'hospitalized' status.

  5. Clean column sex by replacing NA or empty values with 'other' and junk values like 4000 with 'other.'

Read in the file¶

Downloading two files:

  1. covid_data.csv: one file contains age, symptoms, chronic diseases and other infromation. We will use this file to gather information on the age group affected, and the symptoms experienced by infected population.

  2. covid_outcome.csv: contains data like Province/State Country/Region Last Update Confirmed Deaths Recovered This file is more suited to do analysis on overall cases, deaths, recoveries in different countries.

def age_range_avg(x):
    if type(x) == str:
#         Due lack of time using hard coded values for dates to age conversion 
        if x.find('Aug') != -1:
            x = 51.7
        elif x.find('Oct') != -1:
            x = pd.np.nan
        elif x.find('May') | x.find('May') != -1:
            x = 60.1
        elif x.find('Jan') != -1:     
            x = 0.33        
        elif x.find('-') != -1:
#             For values like 30-40 calculate avg using 30 + 40 / 2
            ages = x.split("-")
            ages = pd.to_numeric(ages)
            avg = 0
            for age in ages:
                avg = avg + age
            avg = avg / len(ages)
            x = avg
    if pd.isnull(x):
        x = 0
    return x

  3. Similary for difficulty in breathing, symptoms column contains values with different wordings. For analysis we are just looking at 'difficulty breathing' as a symptom, if it contains words 'breath'. Points 2 & 3, done in custom function named 'generalize_symptoms'.

  5. Inserting a new column 'symptoms_binary'. It contains a '1' if 'symptoms' column was not null for that record. This will ease in analysis later. Point 5 is done in custom function name 'add_col_for_every_symptom'.

def get_individual_symptoms(symp):    
    arr1 = []   
    
    for x in symp:
        if type(x) == str:
            if x.find(',') != -1:
                a = x.split(',')
                arr1 = arr1 + a
            else:
                arr1.append(x)
    return arr1

# 2. For fever, symptoms column contains values like fever, low/mild fever, fever (XX degree celsius).
# 3. Similary for difficulty in breathing, symptoms column contains values with different wordings.
def generalize_symptoms(arr1):
    arr2 = []
    for x in arr1:
        x = str.strip(x)
        x = str(str.lower(x))
        if x.find('breath') != -1:
            x = 'difficulty breathing'
        elif x.find('fever') != -1:
            x = 'fever'
            arr2.append(x)
        elif x == 'symptoms' or x =='muscle' or x=='':
            x = x
        else:
            arr2.append(x)
    return arr2
